# 镀金笼子 - 详细大纲

## 第一幕：黄金时代（第1-100章，约30万字）

### 第一卷：春天的开始（第1-25章）
**时间**：初二下学期（3月-6月）
**基调**：明媚、青涩、纯真

#### 核心情节
- **第1章：钢琴与阳光** - 林晚晚在琴房弹琴，江晨第一次出现在门口
- **第2章：转学生** - 江晨转入班级，两人第一次正式交谈
- **第3章：午后雨** - 共用一把伞回家的路上
- **第4章：钢琴比赛** - 晚晚获奖，江晨送她的第一份礼物（一只纸折的蝴蝶）
- **第5章：图书馆的午后** - 一起学习的日常
- **第6章：第一次牵手** - 在学校天台看夕阳
- **第7章：偷来的时光** - 放学后的小公园
- **第8章：他的秘密** - 发现江晨独自流泪
- **第9章：约定** - "等我们长大了，一起去北京"
- **第10章：初夏的风** - 校园运动会
- **第11-20章**：日常的甜蜜，微小的成长
- **第21章：变化的开始** - 江晨的"叔叔"第一次出现
- **第22章：礼物** - 昂贵的新钢琴
- **第23章：第一次邀请** - "暑假来我家玩吧"
- **第24章：不安** - 晚晚的直觉警告她
- **第25章：出发** - 坐上了去往未知的车

#### 情感弧线
从完全的纯真到微妙的不安。爱情是完美的，但完美的背后开始出现裂痕。

## 写作原则

1. **隐喻优于直白**：用光线、气味、声音暗示
2. **心理描写**：聚焦内心世界，而非外部行为
3. **对比手法**：美好与残酷的并置
4. **感官细节**：让读者"感受到"而非"看到"
5. **留白**：最黑暗的部分永远在字里行间之外
6. **节奏控制**：美好时缓慢如蜜，痛苦时急促如刀
