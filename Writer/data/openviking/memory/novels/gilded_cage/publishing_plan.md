## 简介（Blurb）

### 中文版
```
十四岁的林晚晚以为自己只是遇到了一个特别的男孩。

那阳光倾泻的午后，钢琴声在老楼里回响，他走进她的生命，
说出一句她无法理解的话："你弹琴，缺的是痛。"

她不知道，那是一扇通往深渊的门。

从普通的江南小城，到光鲜亮丽的富人圈。
从纯真的校园初恋，到她无法逃离的金色牢笼。
从最极致的美好，到最深沉的黑暗。

这是一个关于纯真如何被摧毁的故事。
关于那些被隐藏在光鲜表面下的秘密。
关于一个女孩如何在深渊中寻找自我。

--------------------
⚠️ 内容警告：本文涉及未成年人心理创伤、权力不对等关系等敏感主题。
所有黑暗内容以隐喻和心理描写方式处理。
基于真实社会问题改编。

"世界上最美丽的东西，往往也是最危险的。"
```

### English
```
At fourteen, Lin Wanwan thought she had simply met a special boy.

That afternoon when sunlight cascaded through the windows and piano notes 
echoed through the old building, he walked into her life and spoke words 
she couldn't understand:

"You play beautifully. But you're missing something. You're missing pain."

She didn't know then that those words were a door leading into an abyss.

From an ordinary city in southern China to the glittering world of the elite.
From an innocent school romance to a golden cage she cannot escape.
From the most exquisite beauty to the deepest darkness.

This is a story about how innocence is destroyed.
About the secrets hidden beneath gilded surfaces.
About a girl's search for herself in the depths of despair.

--------------------
⚠️ Content Warning: This story addresses sensitive themes including 
psychological trauma of minors and power-imbalanced relationships.
All dark content is handled through metaphor and psychological nuance.
Inspired by real-world issues.

"The most beautiful things in this world are often the most dangerous."
```

## 翻译工作流 / Translation Workflow

### 翻译原则
1. **文学性优先**：不是字面翻译，而是情感翻译
2. **文化适应**：保留中国背景，但让西方读者能理解
3. **一致性**：术语、人名、地名保持统一

### 术语对照表 / Terminology
| 中文 | English | Notes |
|------|---------|-------|
| 林晚晚 | Lin Wanwan | Protagonist |
| 江晨 | Jiang Chen | Male lead |
| 沈夫人 | Madame Shen | Antagonist figure |
| 白露 | Bai Lu | Supporting character |
| 镀金笼子 | The Gilded Cage | Title/Metaphor |
| 庄园 | The Manor | Key setting |
| 圈子 | The Circle | Secret society |

### 翻译工具
- 初稿：人工翻译（保持文学性）
- 校对：AI辅助（检查遗漏）
- 最终：人工审校

## 使用发布器 / Using the Publisher

### 首次设置
```bash
# 设置Wattpad凭据
export WATTPAD_USER="your_email@example.com"
export WATTPAD_PASS="your_password"

# 设置Royal Road凭据
export ROYALROAD_USER="your_email@example.com"
export ROYALROAD_PASS="your_password"
```

### 发布到Wattpad（中文）
```bash
python -m src.main publish \
    --platform wattpad \
    --novel-id gilded_cage \
    --create-story \
    --chapter 1 \
    --language zh
```

### 发布到Wattpad（英文）
```bash
python -m src.main publish \
    --platform wattpad \
    --novel-id gilded_cage_en \
    --create-story \
    --chapter 1 \
    --language en
```

### 发布到Royal Road（英文）
```bash
python -m src.main publish \
    --platform royalroad \
    --novel-id gilded_cage_en \
    --create-story \
    --chapter 1
```

### 批量发布双语版本
```bash
# 发布同一章节到所有平台
python -m src.main publish \
    --platform all \
    --novel-id gilded_cage \
    --chapter 1
```

## 备注 / Notes

这部作品的核心价值在于：
1. **文学性**：隐喻和象征，而非直白描述
2. **情感真实**：让读者真正感受到美好和痛苦
3. **社会意义**：揭露现实中存在但被忽视的问题
4. **尊重受害者**：不是消费痛苦，而是为沉默者发声

The core values of this work:
1. **Literary Quality**: Metaphor and symbolism, not explicit description
2. **Emotional Truth**: Let readers truly feel beauty and pain
3. **Social Significance**: Expose real but overlooked issues
4. **Respect for Survivors**: Not consuming pain, but speaking for the silent

---

*最美的东西最危险，最痛的故事最需要被讲述。*
*The most beautiful things are the most dangerous; the most painful stories are the ones that most need to be told.*
