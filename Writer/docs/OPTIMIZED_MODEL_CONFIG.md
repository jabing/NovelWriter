## 🎯 质量保证策略

### 绝不妥协的质量 (Always Premium)

以下场景**无论成本**都使用最高质量模型：

1. **黄金三章** (第1-3章)
   - 决定读者是否继续阅读
   - 模型: Kimi K2.5 / Claude 3.5
   - 评审: 必须达到9.0分

2. **情感高潮**
   - 表白、冲突、牺牲等关键情感
   - 模型: Kimi K2.5 / Claude 3.5
   - 要求: 情感细腻、层次分明

3. **故事高潮** (85-95%章节)
   - 决定读者满意度
   - 模型: Kimi K2.5 / Claude Opus-4
   - 要求: 张力十足、逻辑严密

4. **大结局**
   - 决定口碑和追更率
   - 模型: Kimi K2.5 / Claude Opus-4
   - 要求: 圆满或震撼、不烂尾

### 可以降级的场景 (Smart Downgrade)

1. **日常推进章节**
   - 剧情平稳发展期
   - 降级为: DeepSeek / GPT-4o Mini
   - 质量损失: <10% (可接受)

2. **环境描写**
   - 场景铺垫
   - 降级为: GPT-4o Mini / Haiku
   - 质量损失: <15% (可接受)

3. **系统/数据展示**
   - 游戏化小说中的属性面板
   - 降级为: GLM-5 (FREE)
   - 质量损失: 0% (甚至更好)
